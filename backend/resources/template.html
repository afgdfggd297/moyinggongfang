s ( x , l ) = f ( x ) ⊗ w ( l )
x暴露浓度
l滞后天数
f(x)暴露-反应基函数（ns, df=3）
w(l)滞后-效应基函数（ns, df=3, 0~7d）
⊗张量积 → 二维效应面
优势：不用为每个滞后天数单独建模
整张效应面一次拟合完成
模型方程与参数体系
5 / 15
Log[E(Yt)] = α + cb(S, lags) + ns(其他污染物, df) + ns(气象, df) + ns(时间, df) + DOW
方程项角色本案例取值
Log[E(Yt)]连接函数Log（线性预测值 → 正值）
α截距项基线死亡风险
cb(S, lags)→ 核心交叉基 ←暴露维 ns(df=3) × 滞后维 ns(df=3, lag 0~7)
ns(其他污染物)控制共存污染物混杂排除 rs > 0.6 的变量
ns(气象)控制气象混杂温度+气压+湿度+风速，ns(df=3)
ns(时间)去除长期趋势和季节性ns(df=6/年)，AIC=10,753 最优
DOW控制一周逐日波动星期哑变量（周一至周日）
建模提示： PM2.5 和 PM10 分别建模（rs=0.845） ｜ 排除 rs > 0.6 的共线变量 ｜ Quasi-Poisson 处理过度离散
分布选择：Quasi-Poisson  ｜  CVD 每日死亡是小概率事件，数据存在过度离散（φ > 1）

模型构建实操 —— 参数选择与 AIC 优化
8 / 15
df 选择的权衡

df 太小 → 欠拟合，残留季节趋势
df 太大 → 过拟合，吃掉暴露信号
AIC 最小 → 平衡拟合与复杂度
滞后结构确定

• 暴露-反应维 df=3
　允许适度非线性弯曲
• 滞后维 df=3, lag 0~7d
　覆盖即时到一周窗口
• 评估 单日滞后（lag0~7）
　和 累积滞后（lag01~07）
